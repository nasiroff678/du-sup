# DU-SUP — Design System

## Цветовая палитра

| Переменная | HEX | Назначение |
|------------|-----|------------|
| `primary` | `#007BFF` | Основной цвет (кнопки, ссылки, акценты) |
| `secondary` | `#4A6741` | Вторичный цвет (зелёный, бейджи, бэйджи) |
| `accent` | `#FACC15` | Акцентный жёлтый (CTA кнопки, выделение) |
| `river-deep` | `#1A1A1B` | Основной текст, тёмные элементы |
| `surface-container` | `#F5F7F5` | Фон секций |
| `on-surface-variant` | `#64748B` | Вторичный текст |

## Типографика

- **Заголовки:** Montserrat (extrabold, bold)
- **Основной текст:** Inter (regular, medium, semibold)
- **Mono/лейблы:** JetBrains Mono, Montserrat uppercase

## Ключевые элементы

### Навигация
- Прозрачная при загрузке, становится `bg-white/80 backdrop-blur-md` при скролле
- Логотип SVG (SUP-борд + волны)
- Кнопки: Направления, Сплавы, Прокат, Сапмараны, Йога, Сертификаты, Галерея

### Hero секция
- Видео-фон `/hero-video.mp4` с параллаксом при движении мыши
- Оверлей с туманным эффектом (`hero-mist-overlay`)
- Fade-out вниз для плавного перехода к контенту

### Средние страницы (`midpage-background`)
- Фон: `#FAFBFA` с мягкими radial gradients (зеленовато-белый)
- Анимированные CSS-частицы (водные капли) — 35 штук, всплывают снизу вверх
- Псевдоэлемент `::before` с radial gradient для глубины

### Секции
1. **О нас** — двухколоночная сетка, перекрывающиеся изображения
2. **Услуги** — 4 карточки с иконками Material Symbols
3. **Сплавы** — карточки туров с фото, ценами и кнопками бронирования
4. **Интерактивная карта маршрутов** — компонент `InteractiveRouteMap`
5. **Квиз** — опросник с промокодом
6. **Прокат** — двухколоночная сетка с прайсом
7. **Сапмаран** — синий фон (`bg-primary`), жёлтые акценты
8. **Йога** — песочный фон, круглое фото, цитата
9. **Как это работает** — 4 шага с линией-коннектором
10. **Галерея** — интерактивный просмотрщик
11. **Прайс-таблица** — полная таблица услуг
12. **Подарочный сертификат** — варианты на сумму (1000/2000/3000₽)
13. **FAQ** — аккордеон с вопросами
14. **Безопасность** — 4 правила с иконками

### Футер
- Видео-фон `/footer-video.mp4` (скорость 0.4x)
- Тёмный оверлей `bg-black/55`
- 4 колонки: Описание, Навигация, Полезное, Контакты

## Анимации

- **Framer Motion (motion/react):** появление секций при скролле, hover-эффекты, параллакс hero
- **CSS:** scroll-pulse индикатор, badge-shimmer, водные частицы (`water-float`)
- **Hero:** cursor parallax через `useMotionValue` + `useSpring`
